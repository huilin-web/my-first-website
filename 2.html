CSS

CSS (Cascading Style Sheets)는 웹 페이지의 스타일과 레이아웃을 지정하기 위해 사용되는 스타일시트 언어입니다. HTML이 웹 페이지의 구조와 내용을 정의한다면, CSS는 그 콘텐츠의 외형을 꾸미는 역할을 합니다. CSS를 사용하면 글꼴, 색상, 여백, 테두리, 위치, 배경 이미지 등 다양한 스타일 요소를 설정할 수 있습니다. CSS는 여러 스타일 규칙을 정의하고, 이러한 규칙을 HTML 요소에 적용함으로써 일관된 디자인을 유지할 수 있게 해줍니다.
